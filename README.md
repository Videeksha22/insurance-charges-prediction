# Insurance Charges Prediction

This project predicts medical insurance charges using machine learning.

## Dataset
Insurance dataset containing age, BMI, smoker status, region, and charges.

## Model Used
- Linear Regression

## Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Result
Smoking status has the highest impact on insurance charges.

---

> **Note:** This project focuses on data preprocessing, exploratory data analysis (EDA), and feature understanding.
